## oss-audit-24BSA10295
Open Source Software Audit Project (Linux Based)

## Project Overview

This repository contains my Open Source Software Audit project completed as part of the OSS course at VIT Bhopal. The goal of this project is to understand how open-source software operates in a Linux environment and to explore its philosophy, ecosystem, and structure.

For this audit, I selected **Python** as the open-source software. Python is widely used due to its simplicity, readability, and strong community support.

In this project, I analyzed Python’s presence in a Linux system and developed shell scripts to examine system details, software installation, file permissions, logs, and open-source concepts.

---

## Selected Open Source Software

- **Software Name:** Python  
- **Category:** Programming Language  
- **License:** Python Software Foundation License  

Python was created by Guido van Rossum in 1991. It focuses on code readability and developer efficiency. Today, it is widely used in:

- Software Development  
- Data Science  
- Artificial Intelligence  
- Web Development  
- Automation & Scripting  

Its open-source nature allows developers worldwide to contribute and improve it continuously.

## Script Descriptions

### Script 1 – System Identity Report
Collects basic system details such as:
- Current user  
- Linux distribution  
- Kernel version  
- Uptime  
- Date & time  

---

### Script 2 – FOSS Package Inspector
Checks Python installation:
- Detects if Python is installed  
- Displays version  
- Shows brief information  

---

### Script 3 – Disk & Permission Auditor
Analyzes important directories:
- `/etc`, `/var/log`, `/home`, `/usr/bin`, `/tmp`  
- Shows permissions, ownership, and size  

---

### Script 4 – Log File Analyzer
Monitors log files:
- Searches for keywords like "error"  
- Counts occurrences  
- Displays recent entries  

---

### Script 5 – Open Source Manifesto Generator
Interactive script that:
- Takes user input  
- Generates a short open-source statement  
- Saves it as a text file
